The Essentials of Responsive Web Design

Essentially, I'm already a bit ticked off. The author acts as if you already have a really strong grasp of github, and that you'll automatically know that the html file is going to be titled "index", for some reason.

  • saying "max-width:100%;" means that an image will grow to be a maximum of 100% of their size, based on the space available.
  • saying "width:100%" means that an image will take up 100% of the available space, even if that means the image must be increased beyond its native size.
Breakpoint
A "breakpoint" is a viewport width or height where a responsive design should change significantly.

Sometimes people create breakpoints with certain devices in mind. This is a bad practice. Instead, these breakpoints should center around when content starts to look bad.

"The absence of support for media queries is in fact the first media query."

Basically, our website should default to the mst basic rules for the most basic screens and devices.

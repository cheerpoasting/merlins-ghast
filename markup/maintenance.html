What is this Page?

Merlin's Ghast is a static website, which means that it's made of only HTML and CSS -- it doesn't use any javascript or anything else that can dynamically generate content.

Since the site doesn't automatically update when content is changed, I need to do it by hand. Changing dozens of pages multiple times takes a long time. This page reminds me to go through all of the pages and make sure they're updated.

I really like music, but I don't really spend as much time listening to it as I would like, and I usually have no one to talk to about it. For that reason, I thought I would go ahead and talk about it here.

I am going to try to create a rating system which is a little bit different than usual. What I'm going to do is to first rate 0-10 how much I like it, then I'm going to rate 0-9 how much I think other people might like it. It will be represented in the form of a decimal. So if I hate something but it seems like others would love it, it would get rated 0.9, and if I love it but I think everyone else will hate it, I'll rate it 10.0. This makes the maximum possible score a 10.9 but I think that's basically never going to happen.

I think I'll try to score things like this:
